# hayaosi - スマホで使える早押しボタン

学校の友達同士（最大15人）で、現実の早押しクイズに使える早押しボタンWebアプリ。

▶ **使う**: https://guttyanneruuuuuu.github.io/hayaosi/

## 特徴

- **インストール不要**: ブラウザだけで動く（スマホ・PC両対応）
- **低遅延**: WebRTCデータチャネルでP2P通信
- **公平な判定**: 出題者の端末で受信時刻を一元判定するので「誰が一番早かったか」が明確
- **シンプル**: スタートボタン無し。常に押せる状態
- **白背景 + 定番デザイン**: ゴテゴテしない見やすいUI

## 使い方

1. **出題者** が自分の名前と合言葉（例: `3年A組`）を入れて「**部屋をつくる**」をタップ
2. **回答者** が同じ合言葉を入れて「**部屋に入る**」をタップ（最大14人）
3. 問題が出されたら一番に押した人が出題者画面のトップに表示される
4. 出題者が「**リセット（次の問題へ）**」を押すと次の問題が始められる

## 仕組み

- **PeerJS** (WebRTC) で全クライアントがホスト端末に直接接続
- 押した瞬間にホストへ `buzz` パケットを送信
- ホスト側 `performance.now()` の受信時刻で順位を決定（単一基準なので公平）
- 押下UI・音は送信前に即時再生（クライアント側の体感遅延ゼロ）

## 注意

- 出題者がブラウザを閉じると部屋は消えます
- 同じ合言葉の部屋を同時に2つは作れません
- 企業ファイアウォール等でWebRTCが遮断されている環境では動きません

## 音源

サウンドは [効果音ラボ](https://soundeffect-lab.info/) 風のクイズボタン音を同梱しています。
